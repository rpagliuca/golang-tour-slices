# About

Custom solution for "Exercise: Slices" of "A Tour of Go" (https://tour.golang.org/moretypes/18)

# Instructions

`go build .` 
`./golang-tour-slices`

The output will be a base64 encoded image.

# Extra tool for previewing

You can preview the image in your system with:

`bash preview.sh`
